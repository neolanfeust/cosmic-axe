# Cosmic Axe

A retro-style space shooter game built with HTML5 Canvas.

**Play it live:** https://neolanfeust.github.io/cosmic-axe/

## Features

- Wave-based enemy combat with shields, bombers, and boss encounters
- Axe-throwing mechanic with upgrade paths
- Player ship upgrades: speed, firepower, spread shots
- Asteroid hazards and scoring system
- Fully self-contained single HTML file — no dependencies
